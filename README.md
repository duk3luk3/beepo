# Beepo

A basic morse code trainer intended for experimentation

# Install

Recommended: Setup a virtualenv

```
virtualenv .venv
. ./.venv/bin/activate
```

Install requirements:

```
pip install -r requirements.txt
```

# Run

```
python test.py
```
